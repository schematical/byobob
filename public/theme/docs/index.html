File Upgrades

File changes needed to get your project from Base Admin v1.0 to v2.0

File Updates

Copy the following directories to your project and overrite any necessary files.

/css/
/js/
/img/
/font/

Twitter Bootstrap

Insert the following css Bootstrap files to your css directory.

/css/bootstrap.min.css
/css/bootstrap-responsive.min.css

Insert the following javascript Bootstrap file to your js directory.

/js/libs/bootstrap.min.js

Base Admin 2.0 Styles

Remove the old style sheets from your document.

<link href="./css/base-admin.css" rel="stylesheet">
<link href="./css/base-admin-responsive.css" rel="stylesheet">


Insert the following two stylesheets to the head of your document after the Twitter Bootstrap includes:

<link href="./css/base-admin-2.css" rel="stylesheet">
<link href="./css/base-admin-2-responsive.css" rel="stylesheet">

jQuery

Remove the old jQuery include from your document.

<script src="./js/jquery-1.7.2.min.js"></script>


Insert the following jQuery include with the following in your document above the closing </body> tag:

<script src="./js/libs/jquery-1.9.1.min.js"></script>

jQuery UI

Insert the following to the </head> of your document:

<link href="./css/ui-lightness/jquery-ui-1.10.0.custom.min.css" rel="stylesheet">

Insert the following to the end of your document above the closing </body> tag:

<script src="./js/libs/jquery-ui-1.10.0.custom.min.js"></script>

Font Awesome

Insert the following Font Awesome css files to your css directory overwriting the old file.

/css/font-awesome.min.css

Copy all the files from the /font directory included in this theme to your project.

/font/FontAwesome.otf
/font/fontawesome-webfont.eot
/font/fontawesome-webfont.svg
/font/fontawesome-webfont.ttf
/font/fontawesome-webfont.woff

Form Validation

Example validation rules for you to get started.

Validation Demo

Validation

For complete information about this plugin and how to get started view:

jQuery Validation Documentation

Includes

Insert the following javascript file into your document.

<script src="./js/plugins/validate/jquery.validate.js"></script>

Validation Example

In order to use the validation styles provided with this theme you need combine your written validation rules with the custom rules provided by this theme. An example is below:

var rules = {
	rules: {
		name: {
			minlength: 2,
			required: true
		},
		email: {
			required: true,
			email: true
		},
		validateCheckbox: {
			required: true,
			minlength: 2	
		}
	}
};
	
var validateObj = $.extend (rules, Theme.validationRules);
    
$('#myForm').validate(validateObj);

FAQ Plugin

Easy to use faq plugin with automatic table of contents and in-page search functionality.

FAQ Demo

Includes

Insert the following includes into your document.

<link href="./js/plugins/faq/faq.css" rel="stylesheet" />
<script src="./js/plugins/faq/faq.js"></script>

FAQ Html

The html for the faq plugin is very basic. Styling and layout will be added when the plugin is called.

NOTE: Be sure the set your faq title as an h4 tag as that is what the table of contents is generated from.

<ol class="faq-list">
						
	<li>
		<h4>FAQ Title</h4>
		<p>FAQ Text</p>		
	</li>
	
</ol>

Faq Javascript

Below is the javascript to initialize the faq plugin and the available configuration options

$('.faq-list').goFaq ({ 
	enableSearch: true,
	enableToc: true,
	enableStyling: true
});

Cirque Plugin

Cirque Demo

Setup

Insert the following includes into your document: 

<link rel="stylesheet" href="./js/plugins/cirque/cirque.css">	
<script src="./js/plugins/cirque/cirque.js"></script>


Create an empty div element to hold your stat:

<div class="my-stat"></div>

Attach the Cirque stat widget to your html element: 

$('.my-stat').cirque ({
	value: 234,
	total: 765,
	arcColor: '#FF9900'	
});

Helper

To create a Cirque stat without using writing javascript add the following class to your container: ui-cirque

<div class="ui-cirque" data-value="84"></div>

This will initialize your cirque stat widget with default parameters defined in Theme.js

HTML5 Data Attributes

By setting HTML5 data attributes on a Cirque container you will override any default values previouly set. ALL values of a Cirque widget can be set via data attributes.

<div class="ui-cirque" 
	data-value="2875" 
	data-total="3245" 
	data-arc-color="#4074AA" 
	data-radius="60" 
	data-line-width="10" 
	data-label="ratio">
</div>

<div class="ui-cirque" 
	data-value="13" 
	data-arc-color="#BBBBBB" 
	data-radius="60" 
	data-line-width="10">
</div>

Options

Class Default Description
value 0 Used to calculate the percentage to fill. The 'value' will be divided by the 'total' to derive the percentage.
total 100 Set the total value that will be used in the calculation to derive the percentage.
radius 60 Sets the radius of the circular stat. To get the total width of the canvas multiply the radius by 2.
label percent Set to percent to display the percentage in the stat.
Set to ratio to display the actual ratio used to calculate the stat.
lineWidth 10 Defines the thickness of the circle.
arcColor #0066CC Defines the fill color of the arc.
trackColor #EEEEEE Defines the color of the track.
trackFill transparent Defines the fill color of the track.
animate true Set to true/false depending on animation preference.
speed 20 Set the speed of the animation in milliseconds.
